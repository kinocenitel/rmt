## [1.2.0] - 2026-04-19

### Добавлено
- **Клавиатурная навигация в фильтрах:** поддержка `↑↓` (навигация), `Enter` (выбор), авто-фокус при открытии. Работает для фильтров по уровням и тегам

### Улучшено
- **UI-иконки:** эмодзи и текст во всех кнопках заменены на SVG-иконоки
- **Свёрнутая матрица:** кнопки тулбара в свёрнутом режиме теперь отображаются без текста — только иконки
- **Курсор для заблокированных кнопок:** добавлен `cursor: not-allowed` для задизейбленных элементов

### Исправлено
- **Обход лимита рисков** ([TASK-001](https://github.com/kinocenitel/rmt/issues/1)): исправлена ошибка, позволявшая создать более 100 рисков за счёт архивации и восстановления старых записей. Теперь лимит считается по сумме всех рисков в проекте
- **Drag & Drop** ([TASK-002](https://github.com/kinocenitel/rmt/issues/2)): заблокировано перетаскивание рисков в режимах сортировки, при наличии одного риска и когда активны фильтры

### Тесты и CI
- **Инфраструктура:** Python + Playwright, настроен автопрогон в GitHub Actions при push или PR в main
- **Покрытие:** написано 48 основных тестов (риски, фильтры, сортировка, теги, экспорт, темы)
- **Отладка:** при падении сохраняются скриншоты и видео в артефактах CI
- **Инструменты:** в разметку добавлены `data-testid` для стабильных локаторов, также добавлен бейдж со статусом тестов в README.md

## [1.1.0] - 2026-03-29

### Добавлено

#### Компактный режим матрицы:
- Кнопка в левом верхнем углу (в шапке) скрывает сетку матрицы
- В компактном режиме форма добавления и список рисков разделены на 2 блока
- Количество рисков по уровням вынесено в цветные бейджи (в шапке)
- Выбранный режим сохраняется между сессиями

#### Горячие клавиши:
- Полное управление с клавиатуры (независимо от раскладки)
- Список всех комбинаций — `Shift + ?` или `Shift + /`

--Основные:--
`Shift + N` — новый риск
`Shift + M` — показать/скрыть сетку матрицы
`Shift + T` — переключить тему
`Shift + A` — переключить вкладку (Активные ↔ Архив)
`Shift + S` — сортировка (по кругу)
`Shift + F` — открыть фильтры
`Shift + R` — сброс активных фильтров
`Shift + I` - импорт Excel
`Shift + E` - экспорт Excel
`Shift + P` — экспорт PNG
`Shift + H` — экспорт HTML
`Shift + Del` — очистить список рисков

### Улучшено

#### Доступность и навигация:
- Навигация в dropdown через `↑↓` + `Enter`
- Tab-ловушка в модалках
- Импорт через `<label>` теперь доступен с клавиатуры

#### Переключатель темы:
- Замена на слайдер и вынос в шапку - правый верхний угол

#### Модалка редактирования риска:
- Увеличена ширина окна: 360 → 520px

#### Адаптивность:
- Унифицированы брейкпоинты
- Минимальная ширина: 320px

#### Футер:
- Фиксация снизу (тепер не прыгает)

#### Лимиты и валидация:
- До 100 активных рисков на матрицу
- Название тега до 25 символов
- Ошибки валидации единообразны + автоматическое скрытие

#### Визуальные изменения:
- Обновлён фавикон (логика уровней риска)
- Иконка переключения темы: месяц → месяц со звёздами

### Исправлено
- Фильтр в архиве отключается при 0–1 риске
- Поле тега не очищается при ошибке по длине + добавлен счётчик символов
