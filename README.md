# Risk Matrix Tool

Простой и удобный инструмент для приоритизации рисков на проектах.

Помогает командам быстро понять:
- что критично
- что можно отложить
- где теряется фокус

---

## 🧠 Why

Команды часто тратят время не туда, потому что не понимают, какие риски действительно критичны. Матрица рисков помогает решить эту проблему, экономя время и бюджет.

---

## 💼 Use cases

- QA — приоритизация тест-кейсов и багов
- PM — планирование
- Developer — фокус на критичных зонах

## ⚡ Features

- Создание, редактирование и удаление рисков
- Сортировка рисков по уровню критичности
- Фильтрация рисков по score и тегам
- Быстрое выявление «красных» зон
- Наглядная и понятная визуализация

---

## 🛠 How to use

1. Откройте файл `index.html`
2. Добавьте риски
3. Отсортируй или отфильтруй список
4. Сфокусируйся на критичных
5. При необходимости выгрузите полученный результат
